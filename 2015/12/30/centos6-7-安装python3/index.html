centos6.7 安装python3

python3 安装

  • 下载python
    wget http://www.python.org/ftp/python/3.5.1/Python-3.5.1.tgz

  • 安装环境
    yum install gcc
    yum install zlib-devel
    yum install make
    yum install readline-devel

  • 解压,编译,安装
    tar -zxvf Python-3.5.1.tgz
    cd Python-3.5.1
    ./configure -prefix=/opt/python3
    make all
    make install

    清除编译出的可执行文件和配置文件
    make clean
    清除所有生成文件
    make distclean

  • 查看python3版本
    /opt/python3/bin/python3 –version

  • 做个软连接到当前用户的bin目录
    ln -s /opt/python3/bin/python3 ~/bin/python3

  • 安装好新版本python后,默认的依然是python2.6
    将文件头部的#!/usr/bin/python 修改为 #!/opt/python3/bin/python3 即可

遇到的问题

  • SecureCRT环境下 python3出现按键映射错误问题
    按backspace, 方向键时会出现^H^[[C之类的乱码
    原因是 python是自己编译的, 在编译时配置脚本没有检测到readline开发文件, 所以它没有编译readline模块,
    解决方法: yum install readline-devel 然后重新编译安装python即可
